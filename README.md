# Getting Started with Create React App

Get started

1. 確保本地端也安裝node.js和npm  
2. 使用專案: 下載.zip檔或使用指令: 
```
git clone https://github.com/Kaijun-Huang/Alpha-Shop.git
```
3. 使用terminal進入專案資料夾使用指令:
```
$ npm install 
```
4. 安裝完成後即可啟動專案:
```
$ npm start
```
可於http://localhost:3000/確認專案是否成功啟動  
在terminal使用 ctrl + c 停止專案
